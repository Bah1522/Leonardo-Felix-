Documentos Necessários

Prepare-se com antecedência. Aqui está tudo que você precisa para sua declaração.

Pessoa Física

Documentos Pessoais

✓ RG ou CNH (documento oficial com foto)

✓ CPF do titular e dependentes

✓ CPF do cônjuge (se casado)

✓ Número do Título de Eleitor

✓ Comprovante de endereço atualizado

Rendimentos

✓ Informe de rendimentos de todos os empregadores

✓ Informes de bancos e corretoras

✓ Comprovante de rendimentos do INSS

✓ Recibos de aluguéis (se houver)

✓ Rendimentos de investimentos

Deduções

✓ Comprovantes de gastos com educação

✓ Comprovantes de gastos com saúde

✓ Comprovantes de contribuição ao INSS

✓ Recibos de pensão alimentícia

✓ Doações comprovadas

Bens e Direitos

✓ Escrituras de imóveis

✓ Documentos de veículos (RENAVAM)

✓ Comprovantes de investimentos

✓ Recibos da declaração anterior

Pessoa Jurídica

Documentação Básica

✓ CNPJ da empresa

✓ Contrato social ou estatuto

✓ Inscrição estadual e municipal

✓ CPF dos sócios

✓ Comprovante de endereço da empresa

Documentos Fiscais

✓ Notas fiscais emitidas

✓ Notas fiscais recebidas (compras)

✓ Livros fiscais (Livro de Apuração do ICMS)

✓ Recibos de serviços prestados

✓ Comprovantes de pagamento de impostos

Despesas Operacionais

✓ Comprovantes de aluguel

✓ Contas de água, luz, internet

✓ Comprovantes de folha de pagamento

✓ Recibos de serviços contratados

✓ Comprovantes de manutenção de equipamentos

Informações Financeiras

✓ Demonstrativo de resultado do exercício

✓ Balanço patrimonial

✓ Extratos bancários da empresa

✓ Planilha de faturamento

✓ Comprovantes de pagamento de dividendos
